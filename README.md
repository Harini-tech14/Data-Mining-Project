📌 AI-Driven Career Path Recommendation System
📖 Project Overview

This project is an AI-powered Career Path Recommendation System developed using Data Mining Techniques (DMT) and Machine Learning concepts. The system analyzes users’ skills and interests to suggest suitable career paths along with matching skills and skills to improve.

🚀 Features

Skill-based career matching

Matching skills identification

Missing skills suggestion

Career description modal

Flask backend integration

Dynamic frontend using HTML, CSS, and JavaScript

🛠 Technologies Used

Python

Flask

HTML

CSS

JavaScript

Data Mining Techniques

⚙️ How to Run the Project

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open in browser:

http://127.0.0.1:5000
🎓 Academic Purpose

Developed as part of Data Mining Techniques (DMT) course project to demonstrate real-world application of recommendation systems.
